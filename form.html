2. HTML Validation

The website pages (such as index.html) will be checked using the W3C HTML Validator at https://validator.w3.org/nu .

My goal is to have zero HTML errors reported by the validator. Any warnings will be reviewed and corrected as needed.

3. CSS Validation

The stylesheet (styles/styles.css) will be checked using the W3C CSS Validator at https://jigsaw.w3.org/css-validator/ .

My goal is to have no CSS errors. Any warnings will be reviewed and corrected when appropriate.

4. Site Name

Site Name: Multi-Logistics Ghana

I chose this name because it clearly describes the main focus of the company: providing logistics, shipping, import, and export services in and out of Ghana. It sounds professional and trustworthy to potential clients and businesses.

5. Site Purpose

The primary purpose of the Multi-Logistics Ghana website is to introduce our logistics company and make it easy for customers to:

  • Learn about our shipping, freight forwarding, and port clearance services.
  • Understand how to import vehicles, goods, and containers into Ghana.
  • Request a quote or ask questions using an online contact form.
  • Find our office contact details and operating locations (e.g., Tema Port).

6. Scenarios (User Tasks)

These scenario questions reflect what real visitors to the site might want to do:

  • Scenario 1: “How can I ship a car from the USA to Ghana, and how much will it cost me through Multi-Logistics Ghana?”
  • Scenario 2: “Can Multi-Logistics Ghana help me clear my container at Tema Port and handle all the customs paperwork?”
  • Scenario 3: “Is there a way to contact Multi-Logistics Ghana online to request a quote or talk to a staff member?”

7. Color Scheme

The website will use the following color scheme:

Color Hex Value Usage
Primary (Navy Blue) #0A3760 Header background, navigation bar, headings, and key icons.
Accent (Gold) #F4B400 Call-to-action buttons, highlights, and important links.
Background (White) #FFFFFF Page background and main content areas for good readability.

8. Color Scheme Used in This Document

This planning document already uses the color scheme described above:

  • The header and section titles use the primary navy blue.
  • Important accents, badges, and highlights (like this word highlight) use the gold accent color.
  • The page background is white to match the planned website background.

This shows how the color palette will look on the final website.

9. Typography

The website will use two main fonts:

Font Type Usage
Montserrat Sans-serif (Google Font) Headings, navigation links, and call-to-action buttons.
Arial Sans-serif (web-safe) Paragraph text, lists, and general body content.

10. Typography Used in This Document

This planning document imports Montserrat from Google Fonts and uses it on headings (h1, h2, h3) and some accent elements.

The body text uses Arial, which matches the typography that will be used on the final website pages.

This provides a preview of how the text will look on the finished site.
